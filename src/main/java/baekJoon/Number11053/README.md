## 문제풀이
 1. 숫자 증가가 연속적이지 않아도 됨
 2. 새로운 배열을 생성하고 index가 증가함에 따라 최대 길이를 채워나감
 
> ```
> 입력된 숫자의 배열 array, lenth저장용 lenArray
> - 조건 : 이중 포문에서 i와 j가 잇다고 가정
> - array[i] > array[j] && lenArray[j]+1 > lenArray[i]
> ```

## 예제 입력
6  
10 20 10 30 20 50
